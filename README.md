# Feed-Reader-Testing-Project
This is my application of Feed Reader Testing Project of FEND from Udacity.
# Getting Started
Download the project and open  **index.html** file using any web browser and the testing will be automatically. 
# More Info About Testing 
I used Jasmine library for test some function in the project, the testing is in file **feedreader.js**.
# Built With
* HTML
* CSS
* JavaScript
* [Jasmine](https://jasmine.github.io/) is a library for testing in general ,  I use it for testing the project.
